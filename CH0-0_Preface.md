# 前言 (Preface)

近年來，智慧機器人技術快速發展，應用範圍正逐步由製造場域延伸至醫療、餐飲、物流及巡檢等領域，甚至逐漸成為你我生活中的新夥伴，因此為讓更多人認識它們，經濟部產業發展署與本會（財團法人資訊工業策進會）於2025年率先推出科普讀物《Bot & Build：人機協作到共生的實踐指南》；這本以淺顯易懂的文字搭配資訊視覺化圖像，帶領讀者認識服務型機器人（Service Robot）的發展趨勢與多元應用，並結合近年崛起的智慧建築議題，從建築管理觀點形塑出機器人友善應用文化，期待能在迎接新一波機器人浪潮之前，凝聚社會的期待與支持。

隨著服務型應用需求的日益增加，產業樣態也逐漸由單一設備競爭走向跨系統、跨廠家的整合性合作。然而，目前我國機器人相關產業面臨到零組件供應端與整機需求端之間，仍缺乏一致性的開發與整合共識之問題，多數廠商依循自身技術路線，採用不同的通訊協定、資料格式與驅動套件，導致系統整合、功能擴充與跨廠商協作的成本居高不下，限制了台灣機器人產業生態系的形成；在此背景下，**ROS 2 (Robot Operating System 2)** 因為其高支援特性，逐漸成為服務型機器人領域的主流開發框架，並成為串接軟硬體資源、促進跨廠商合作的重要媒介。

其中，更有感於感知零組件是機器人理解環境、定位導航及自主行動的重要基礎，為協助台灣建立可規模化擴散的機器人技術生態系，我們在主管機關的支持下，於今年（2026）聚焦「**ROS 2 開發框架**」與「**機器人感知零組件**」議題推出第二本技術入門讀物，將從機器人基礎概念出發，引領讀者一步步走進軟硬體整合的開發世界，期待這本指引能成為各行各業踏入機器人開發領域的起點，並支持我國相關零組件商掌握關鍵知識、拓展市場機會，在這波機器人浪潮中能走得更穩、更久，也更遠。

因此，本指引以實務開發流程為基礎，說明依廠牌而異的感測器原始資料，如何透過 ROS 2 框架進行轉換、理解及融合，進而成為機器人可用的感知資訊；此外，亦協助整機廠商在選用零組件時，快速釐清基礎需求對應的技術關鍵字為何，並在供應商提供的軟體套件（例如感測器 SDK，或是工業電腦 BSP）支援下，以較少資源完成概念驗證，而無需耗費大量時間處理驅動、介面或作業系統等底層整合工作；最終，透過較一致的工程思維、開發介面與整合方法之形塑，為國內機器人產業奠定感知技術領域的通用化基礎。指引共分三章，內容綱要如下：
