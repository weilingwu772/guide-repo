## 🤝 如何參與協作

本次文件撰寫工作採 GitHub 搭配 Antigravity 進行，稿件內容不須登入即可閱覽。若要提出調整建議或修改內容，請依有無 GitHub 帳號選擇以下方式：

### 🛠️ 方法一：有 GitHub 帳號
採標準的 **Fork & Pull Request** 流程（可參閱 [GitHub flow](https://docs.github.com/en/get-started/quickstart/github-flow)）：
1. **Fork 本儲存庫**：點擊頁面右上角的 "Fork" 按鈕，將專案複製到您自己的 GitHub 帳號下。
2. **複製到本地**：將您的 Fork 版本複製（Clone）到本地電腦進行修改。
3. **建立新分支**：為您的修改建立一個獨立的特徵分支（例如 `git checkout -b feature/your-topic`）。
4. **提交修改**：撰寫或修改內容後，提交變更並推送至您的 GitHub 儲存庫。
5. **發送 Pull Request**：在 GitHub 頁面上向本儲存庫提交 Pull Request，我會在確認後將其合併。

### ✉️ 方法二：沒有 GitHub 帳號
如果您沒有 GitHub 帳號，也可以透過以下方式傳送給我：
- **複製文字修改**：直接複製欲修改的段落文字，修改後提供「修改前/修改後」的文字。
- **提供畫面截圖**：將需要調整或有誤的畫面截圖，並標註您的建議後提供。
